# Campus Guardian AI

Campus Guardian AI is a production-level computer vision surveillance and safety monitoring system for campus environments.

The system uses real-time video processing, object detection, tracking, face recognition, incident logging, and analytics dashboards to help detect and monitor campus safety events.

## Project Goals

- Real-time webcam/CCTV monitoring
- Person, vehicle, and animal detection
- Intrusion and loitering detection
- Unknown person detection
- Attendance monitoring
- Incident logging
- Analytics dashboard
- Full-stack deployment-ready architecture
